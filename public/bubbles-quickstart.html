Bubbles AI — Quick Start Card

Use these proven phrases and sample request formats to quickly send documents, create tasks, and set reminders via Bubbles chat or voice.

Top Phrases (copy & say)

  1. "Send upfront_warranty_5-year_v4.pdf to Sarah Owner and John Smith with a message saying Please review and sign."
  2. "Attach inspection_report_v2.pdf to Project #12345 and create a task for Jane Doe due in 3 days."
  3. "Open inspection_report_v2.pdf and list the top 10 steps."
  4. "Send the roof packet to the project manager and set a reminder for next Tuesday at 9am."
  5. "Summarize the inspection_report_v2.pdf and extract any action items."
  6. "Send upfront_warranty_5-year_v4.pdf to all internal users and notify them to review."

JSON Request Samples

Chat API: POST /api/bubbles/chat
Send doc + notify + task
{
  "message": "Send inspection_report_v2.pdf to Jane Doe and Carlos Martinez with a message saying Please review the inspection comments. Also create a task to follow up in 3 days.",
  "projectId": "proj_7a2f"
}
Extract steps from doc
{
  "message": "Open inspection_report_v2.pdf and give me the top 10 steps.",
  "projectId": "proj_7a2f"
}

Voice Shortcut

Say this during a call:

"Hey Bubbles, send upfront_warranty_5-year_v4.pdf to Sarah Owner and John Smith and tell them 'Please review and sign.' Also create a task for Sarah to verify in three days."

Do / Don’t

  • Do: Use exact filenames or full user names for best results.
  • Do: Include projectId if you mean a specific project.
  • Don't: Ask to attach files that haven’t been uploaded to Documents & Resources.
  • Don't: Expect external email sends unless you explicitly initiate a share via Documents UI.
