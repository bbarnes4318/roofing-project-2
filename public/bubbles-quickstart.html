Bubbles Assistant Playbook

Clear prompts and voice tips for sending project updates, tasks, and reminders with documents attached. Speak plainly—Bubbles handles the rest.

Prompts to try
  • "Send progress_photos.pdf to the homeowner with a note saying crew wrapped by 4pm."
  • "Create a task for Sarah to order shingles for project 10023 by Friday and attach the takeoff PDF."
  • "Set a reminder for the install crew tomorrow at 8am to upload day-one photos."
  • "Summarize punchlist_notes.pdf and send the highlights to the production manager."
Document tips
  • Name it exactly: "Attach inspection_report_v2.pdf."
  • Need help finding it? "What documents are on project 10045?" then, "Send the roof packet to accounting."
  • Multiple files: "Send the permit.pdf and latest_photos.zip to the homeowner."
  • Just uploaded? Mention it by name—Bubbles checks Documents & Resources instantly.
Voice: natural names first

Speak it normally: "Attach the roofing doc forty-eight thirty-eight F F one PDF." Bubbles handles underscores and dashes automatically.

  • Add context: Mention the project, uploader, or “the warranty PDF from today” to help it choose the right file.
  • Fallback: Only spell it out (“roofing underscore doc…”) if Bubbles asks for clarification.
  • Numbers: If you do spell it, say digits individually (“four eight three eight”).
Reliable workflow
  • 1. State the job: “Send a status update to Maria about today’s roof progress.”
  • 2. Add project & people: Include the project number or names so Bubbles routes it right.
  • 3. Approve the recap: Bubbles repeats the plan—say “Looks good” or make tweaks before it sends.
  • Need a human? Email support@kenstruction.com.
Voice shortcut example

“Hey Bubbles, send roofing underscore doc underscore four eight three eight dash F F one dot P D F to the production team with the message ‘Please review before tomorrow’s install.’ Also set a reminder for me at 7am to confirm they received it.”
